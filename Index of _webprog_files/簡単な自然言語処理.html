1 Mecabによる日本語処理

以下は日本語wikipediaのchatGPTの概要の固有名詞を分析した例である。 このプログラムはEWSのLinuxでは動かない。10.133.2.200のサーバーでは動くのでリモートログインして動かしてみよう。 コメントアウトされているprint文などのコメントを外して、どんな動作をしているのか確認してみよう。 動詞についても同様の分析ができるのでやってみよう。

最後のprint(t,pns.count(t))の部分で固有名詞の出現回数を数えている。 この概要の文章で一番出現回数の多かった単語はChatGPTで次に多かったのがOpenAIであり、 ChatGPTの重要な属性の１つがOpenAIという単語であることがわかる。

このように文章中の単語の出現回数の一番上ではなく、2位以下の適当な上位ぐらいに重要な情報なことがあることが多く、 そのような分析を一般化したものがTf-idfと呼ばれる。 詳細は https://ja.wikipedia.org/wiki/Tf-idf を参照すること

dyn/examples/wikijplxml2.py: 

 1: import sys
 2: import requests
 3: import lxml.html
 4: import MeCab
 5: m = MeCab.Tagger("-Ochasen")
 6: 
 7: url="https://ja.wikipedia.org/wiki/ChatGPT"
 8: 
 9: response = requests.get(url)
10: htmldata = lxml.html.fromstring(response.content)
11: 
12: pns=[]
13: 
14: xp='/html/body/div[2]/div/div[3]/main/div[3]/div[3]/div[1]/p[4]'
15: p_content = htmldata.xpath(xp)
16: for content in p_content:
17:     text=content.text_content()
18:     result = m.parse(text)
19:     proper_nouns = [line for line in m.parse(text).splitlines()  if "固有名詞" in line.split()[-1]]
20:     #print(proper_nouns)
21: 
22:     proper_nouns1=[s.split()[0] for s in proper_nouns]
23:     #print(proper_nouns1)
24: 
25:     pns += proper_nouns1
26: 
27: #sys.exit()
28: 
29: for i in range(5,9):
30:     xp='/html/body/div[2]/div/div[3]/main/div[3]/div[3]/div[1]/p['+str(i)+']'
31:     p_content = htmldata.xpath(xp)
32:     for content in p_content:
33: 	text=content.text_content()
34: 	result = m.parse(text)
35: 
36: 	proper_nouns = [line for line in m.parse(text).splitlines()  if "固有名詞" in line.split()[-1]]
37: 	proper_nouns1=[s.split()[0] for s in proper_nouns]
38: 	#print(proper_nouns1)
39: 
40: 	pns += proper_nouns1
41: 
42: print(pns)
43: 
44: for t in set(pns):
45:     print(t,pns.count(t))

2 NLTKによる英語処理

以下は英語wikipediaのchatGPTの概要の固有名詞を同様に分析した例である。コメントアウトされているprint文などのコメントを外して、どんな動作をしているのか確認してみよう。 このプログラムは動かすにはまず、ライブラリのインストールが必要になる。 EWSのLinuxでは以下のコマンドでライブラリのインストールができる。 

pip install nltk

10.133.2.200のサーバーでは以下のコマンドでライブラリのインストールができる。 

pip install nltk --break-system-packages

使用しているライブラリはNLTKと呼ばれるライブラリで、他にも非常に多くの機能を使うことができる。調べて使ってみよう。

dyn/examples/wikienlxml.py: 

 1: import sys
 2: import requests
 3: import lxml.html
 4: 
 5: 
 6: import nltk
 7: nltk.download('punkt')
 8: nltk.download('averaged_perceptron_tagger')
 9: 
10: url="https://en.wikipedia.org/wiki/ChatGPT"
11: 
12: response = requests.get(url)
13: htmldata = lxml.html.fromstring(response.content)
14: 
15: pns=[]
16: 
17: for i in range(2,5):
18:     xp='/html/body/div[2]/div/div[3]/main/div[3]/div[3]/div[1]/p['+str(i)+']'
19: 
20:     p_content = htmldata.xpath(xp)
21:     for content in p_content:
22: 	text=content.text_content()
23: 	print(text)
24: 	tokenized_words = nltk.word_tokenize(text)
25: 	pos_tagged_text = nltk.pos_tag(tokenized_words)
26: 	#print(pos_tagged_text)
27: 
28: 	proper_nouns = [w[0] for w in pos_tagged_text  if w[1]=='NNP' ]
29: 	print(proper_nouns)
30: 
31: 	pns += proper_nouns
32: 
33: print(pns)
34: 
35: for t in set(pns):
36:      print(t,pns.count(t))

NLTKの他には gensim というライブラリも英語の分析には良く使われる。gensymも使用例を調べてみよう。
