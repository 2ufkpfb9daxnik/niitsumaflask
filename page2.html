1 簡単なホームページ作成

コマンドラインで 

pip install -U flask

と入力してflaskというウエブサイトを動かすライブラリをインストールする。

以下を app.py として(geditなどを起動して)保存する

dyn/examples/flasksimple.py 

1: from flask import Flask
2: 
3: app = Flask(__name__)
4: 
5: @app.route("/")
6: def hello_world():
7:     return "<p>Hello, World!</p>"
8: if __name__ == '__main__':
9:     app.run()

コマンドラインで 

python3 app.py

を実行して、 Linuxのfirefoxで http://localhost:5000/　 を開く 

firefox http://localhost:5000/

自分で動かしたホームページにHello, World!と表示されるのを確認する。

2 html タグについて学ぶ

  • <p> </p>を <h1> </h1>に変えてホームページがどのように変わるか確認する
  • Hello の後ろに<br> を挿入しホームページがどのように変わるか確認する
  • google でhtmlタグと検索して、他にも様々な html タグについて調べて使ってみよう

発展課題
